❌ المشكلة: "Python was not found"

🔧 الحلول:
1. تثبيت Python:
   • اذهب إلى: https://python.org
   • حمل أحدث إصدار Python
   • تأكد من تحديد "Add Python to PATH" أثناء التثبيت

2. استخدام Python من Microsoft Store:
   • افتح Microsoft Store
   • ابحث عن "Python"
   • ثبت Python 3.11 أو أحدث

3. استخدام ملف التشغيل البديل:
   • انقر نقراً مزدوجاً على: تشغيل_بديل.bat

===============================================================================

❌ المشكلة: "Flask غير مثبت"

🔧 الحلول:
1. تثبيت Flask يدوياً:
   • افتح Command Prompt كمدير
   • اكتب: pip install flask
   • اضغط Enter

2. استخدام py:
   • اكتب: py -m pip install flask

❌ المشكلة: "لا يمكن الوصول للموقع"

🔧 الحلول:
1. تأكد من تشغيل النظام بنجاح
