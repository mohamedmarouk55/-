🚀 تعليمات تشغيل نظام الإدارة الشامل 🚀

═══════════════════════════════════════════════════════════════

📋 المتطلبات:
1. تثبيت Python (الإصدار 3.7 أو أحدث)
   - تحميل من: https://www.python.org/downloads/
   - تأكد من تحديد "Add Python to PATH" أثناء التثبيت

═══════════════════════════════════════════════════════════════

🔧 طرق التشغيل:

الطريقة الأولى - استخدام ملف التشغيل التلقائي:
1. انقر نقراً مزدوجاً على ملف "install_and_run.bat"
2. سيقوم بتثبيت المكتبات وتشغيل التطبيق تلقائياً

الطريقة الثانية - استخدام PowerShell:
1. انقر بالزر الأيمن على ملف "run.ps1"
2. اختر "Run with PowerShell"

الطريقة الثالثة - التشغيل اليدوي:
1. افتح Command Prompt أو PowerShell
2. انتقل إلى مجلد المشروع:
   cd "c:\Users\LENOVO\Desktop\مشروع ادارة السيارات-4.HTML"
3. ثبت المكتبات:
   pip install -r requirements.txt
4. شغل التطبيق:
   python app.py

🛠️ حل المشاكل الشائعة:

❌ خطأ "Python was not found":
- تأكد من تثبيت Python
- أعد تشغيل الكمبيوتر بعد التثبيت
- تأكد من إضافة Python إلى PATH

❌ خطأ في تثبيت المكتبات:
- تأكد من اتصال الإنترنت
- جرب: pip install --upgrade pip
- ثم أعد تشغيل الأمر

❌ التطبيق لا يفتح:
- تأكد من عدم استخدام المنفذ 5000 من تطبيق آخر
- جرب إغلاق برامج أخرى قد تستخدم نفس المنفذ
